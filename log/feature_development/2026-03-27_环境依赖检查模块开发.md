## 🔍 详细信息

### 技术实现
- **编程语言**: Python 3.11
- **依赖模块**: subprocess, sys, pathlib, json, datetime
- **输出格式**: Markdown报告文件

### 核心功能
1. **Python版本检查**: 检查Python版本是否满足要求
2. **依赖包检查**: 检查所有requirements.txt中的包是否已安装
3. **NDL OCR检查**: 检查ndlocr-lite是否正确安装
4. **系统信息**: 获取操作系统、内存等信息
5. **配置文件检查**: 检查.env等配置文件

### 使用方法
```python
from modules.environment_checker import run_environment_check

passed = run_environment_check("PDF处理任务")
```
