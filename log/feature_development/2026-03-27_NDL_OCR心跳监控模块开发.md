## 🔍 详细信息

### 技术实现
- **编程语言**: Python 3.11
- **依赖模块**: threading, subprocess, pathlib, datetime
- **心跳间隔**: 300秒（5分钟）
- **告警条件**: 连续失败3次

### 核心功能
1. **心跳检测**: 定期发送测试请求检查服务可用性
2. **状态追踪**: 记录连续成功/失败次数
3. **自动重试**: 失败时自动重试，最多3次
4. **告警机制**: 连续失败3次以上触发告警
5. **健康报告**: 生成详细的监控报告

### 心跳配置
```python
HEARTBEAT_INTERVAL = 300  # 5分钟
ALERT_THRESHOLD = 3       # 连续失败3次触发告警
```

### 使用方法
```python
from modules.ndl_ocr_monitor import NDLOCRHeartbeatMonitor

monitor = NDLOCRHeartbeatMonitor()
monitor.start_monitoring(interval=300)  # 5分钟心跳

# 执行任务...

monitor.stop_monitoring()
```
