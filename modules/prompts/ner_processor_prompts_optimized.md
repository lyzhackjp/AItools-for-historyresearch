## 用户提示词

### [NER_OPT_U001] - 优化版命名实体识别提示词

**标识符**: NER_OPT_U001  
**创建日期**: 2026-03-29

**内容**:

```
请从以下日文史料文本中识别并分类历史专有名词。

【实体类型定义】
- person（人物）：历史人物姓名，如"伊藤博文"、"西郷隆盛"、"大久保利通"、"木戸孝允"
- location（地点）：地理位置，如"東京"、"京都"、"薩摩"、"長州"、"江戸"、"大阪"
- organization（组织）：机构组织，如"幕府"、"内務省"、"外務省"、"貴族院"、"衆议院"
- event（事件）：历史事件，如"明治維新"、"大政奉還"、"廃藩置県"、"西南戦争"
- date（年代）：历史年代，如"1868年"、"幕末"、"明治時代"、"戦国時代"
- work（著作）：文献作品，如"《日本国憲法》"、"《学問のすゝめ》"
- concept（概念）：思想概念，如"君主立憲"、"開国进取"、"文明開化"、"殖産興業"

【日本历史特殊提示】
- 时代背景：明治維新、大政奉還、廃藩置県、西南戦争、自由民権運動
- 重要人物：伊藤博文（初代内閣总理大臣）、西郷隆盛（明治三杰）、大久保利通、木戸孝允、板垣退助
- 地理区域：薩摩（鹿兒島）、長州（山口）、土佐（高知）、肥前（九州北部）、会津（福島）
- 机构组织：幕府（徳川幕府）、藩（大名領地）、省（中央官庁）、華族（貴族階級）

【待识别文本】
{text}

【输出格式（JSON数组）】
[
  {{"text": "实体文本", "category": "实体类型", "confidence": 0.95}},
  {{"text": "实体文本", "category": "实体类型", "confidence": 0.85}}
]

请严格按照JSON格式输出，不要包含其他内容。只输出JSON数组，不要添加解释或说明。
```

**模板变量说明**:
- `{text}`: 待识别的日文史料文本

### [NER_OPT_U003] - 历史实体验证提示词

**标识符**: NER_OPT_U003  
**创建日期**: 2026-03-29

**内容**:

```
请验证以下日文历史实体是否正确，并提供置信度评分。

【已识别实体列表】
{entities}

【历史背景知识】
- 明治維新时期（1868-1912）的著名人物：伊藤博文、西郷隆盛、大久保利通、木戸孝允、板垣退助、坂本龍馬、高杉晋作
- 重要地点：東京、京都、薩摩、長州、土佐、肥前、会津、江戸、大阪
- 主要机构：幕府、藩、内務省、外務省、陸軍省、海軍省、大蔵省、文部省、貴族院、衆议院
- 历史事件：大政奉還（1868）、明治維新、廃藩置県（1871）、西南戦争（1877）、自由民権運動（1880s）

1. 确认实体是否存在
2. 验证实体类型是否正确
3. 提供置信度评分
4. 纠正错误的识别

{{

    {{"text": "实体", "category": "类型", "confidence": 0.95, "valid": true}},
    {{"text": "实体", "category": "类型", "confidence": 0.70, "valid": false, "reason": "实体不存在或拼写错误"}}
  ],
  "overall_accuracy": 0.92

```
